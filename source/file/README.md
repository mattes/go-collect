# File source

Currently supports YAML files only. 
[More example files.](https://github.com/mattes/fugu/tree/v1/examples)

## Example URLs

```
--source=file:///absolute/path/to/file.yml
--source=file://relative/path/to/file.yml
--source=file://../relative/path/to/file.yml
--source=file://file.yml
```
